# 🎓 Campus Buzz

### *A Smart Campus Coordination & Communication Platform*
Video link of Campbuzz : https://drive.google.com/file/d/1kJJkRRiwxCjO1VNApHuq_i6LZRxDAsgv/view?usp=sharing

> **Campus Buzz** is a centralized platform designed to simplify campus communication, event management, student collaboration, and grievance handling through secure role-based access and real-time interaction.

---

## 📖 Overview

Campus communication is often scattered across multiple WhatsApp groups, emails, notice boards, and social media, making it difficult for students, clubs, and administrators to stay connected.

**Campus Buzz** brings everything together into one unified platform where students can interact, discover events, collaborate, report issues anonymously, and receive official announcements from verified campus organizations.

---

## 🚀 Problem Statement

Traditional campus communication suffers from:

* Fragmented information across multiple platforms
* Lack of centralized event management
* Difficulty connecting students with campus clubs
* Inefficient complaint reporting systems
* Poor collaboration among students
* No secure role-based communication

Campus Buzz addresses these challenges with an integrated digital ecosystem for the entire campus community.

# ✨ Features

## 🔐 Secure Authentication

* Institute Email Verification
* Roll Number Based Login
* JWT Authentication
* Protected Routes
* Secure Password Encryption

---

## 👥 Multiple User Roles

The platform supports different user roles with dedicated permissions.

* 🎓 Student
* 🏛 Club / Committee
* 👨‍💼 Administrator

Each role has access only to the features relevant to them.

---

Students can create and interact with campus posts.

* Image Upload

* Search Posts

* Infinite Scrolling Feed
* Automatic Post Expiry
* Countdown Timer

Supported Categories:

* 🍽 Food Split

* 🔍 Lost
* 🎒 Found

# ⚙ Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project directory

```bash
cd ACE-Hackathon
```

Install dependencies

```bash
npm install
```

Start the backend server

```bash
node server.js
```

Open the frontend in your browser.
